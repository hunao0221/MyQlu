# MyQlu
## 应用介绍
  MyQlu是齐鲁工业大学校园客户端，仅限齐鲁工业大学学生使用，部分功能只能在校园网环境下使用
## 现有功能：
- 课程查询
- 成绩查询
- 考试查询
- 一卡通（饭卡）当日流水查询
- 一卡通一键挂失
